## PackMind

This project uses **PackMind**, a committed project memory for Claude Code.
Read `.packmind/PACKMIND.md` once. The brain files (`.packmind/knowledge.md`,
`solutions.json`, `handoff.md`) are shared team memory: keep them current with
the **packmind** MCP tools (`recall` to search, `remember` for durable lessons,
`record_solution` when a fix works, `handoff` before stopping), and heed
guardrail warnings before writing files.
